# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-22

### Added
- **房间系统**
  - 创建公共/私人农场房间
  - 自定义房间名称和大小 (5x5 ~ 20x20)
  - 房间列表展示

- **玩家系统**
  - 玩家名称设置
  - 金币系统 (初始 50 金币)
  - 在线玩家列表

- **种植系统**
  - 三种作物: 小麦、番茄、玉米
  - 种子购买与种植
  - 作物生长阶段 (种子→幼苗→生长中→成熟)
  - 收获功能

- **浇水系统**
  - 手动浇水 (+30% 湿度)
  - 土壤湿度可视化

- **多人同步**
  - 实时位置同步
  - 作物状态同步
  - 协作加成 (多人在线时触发)

- **操作反馈**
  - 移动、浇水、收获动画
  - 地块信息提示 (湿度、生长进度)
  - 操作结果通知

- **界面**
  - 大厅界面 (房间列表、作物图鉴)
  - 游戏主界面
  - 顶部信息栏 (房间名、金币、在线人数)
  - 底部通知栏

### Added (Keyboard Support)
- WASD / 方向键移动
- 空格键浇水

### Technical
- 基于 Express + Socket.io 构建
- WebSocket 实时通信
- PM2 进程管理支持
