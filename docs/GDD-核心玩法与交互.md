# Feng Farm 核心玩法与交互设计

## 1. 游戏定位

**一句话描述：** 多人实时协作种田游戏，打造真实生态系统，支持功能模块化扩展。

**核心体验：**
- 简单易上手：3分钟学会基础操作
- 策略有深度：土壤、天气、生态相互影响
- 社交有乐趣：分工协作、互助交易

---

## 2. 核心玩法循环

```
┌─────────────────────────────────────────┐
│           进入农场（房间）               │
└───────────────┬─────────────────────────┘
                ↓
┌─────────────────────────────────────────┐
│  1. 准备阶段：选择种子、规划种植布局      │
│     - 查看当前土壤状态                  │
│     - 选择合适作物                      │
└───────────────┬─────────────────────────┘
                ↓
┌─────────────────────────────────────────┐
│  2. 种植阶段：播种、浇水、维护           │
│     - 点击空地移动                       │
│     - 选择作物种植（消耗金币）           │
│     - 定期浇水（加速生长）               │
│     - 观察生长进度                       │
└───────────────┬─────────────────────────┘
                ↓
┌─────────────────────────────────────────┐
│  3. 收获阶段：收割、售卖、再投资         │
│     - 作物成熟后收割                     │
│     - 获得金币奖励                       │
│     - 购买更好种子/扩建农场              │
└───────────────┬─────────────────────────┘
                ↓
┌─────────────────────────────────────────┐
│  4. 社交互动：协作、交易、竞争           │
│     - 帮好友浇水（获得奖励）             │
│     - 交易多余作物                       │
│     - 排行榜竞争                         │
└───────────────┬─────────────────────────┘
                ↓
         回到步骤 1，循环往复
```

## 3. 玩家操作流程（详细）

### 3.1 首次进入游戏

```
打开游戏链接
    ↓
输入玩家名称
    ↓
选择加入方式：
    ├─ 加入公共农场（默认房间，快速开始）
    └─ 创建私人农场（自定义房间名和大小）
    ↓
进入游戏主界面
```

### 3.2 核心操作指令

| 操作 | 触发方式 | 效果 | 反馈 |
|------|---------|------|------|
| **移动** | 点击地图空地 | 玩家移动到该位置 | 玩家标记移动到目标地块 |
| **种植** | 站立时点击"种植"按钮，选择作物 | 消耗金币，种下种子 | 地块显示种子图标，金币减少 |
| **浇水** | 站在有作物的地块，点击"浇水" | 土壤湿度+30%，加速生长 | 地块显示水滴动画，作物生长加快 |
| **收获** | 站在成熟作物旁，点击"收获" | 获得金币，地块清空 | 金币增加，显示收获动画 |
| **查看信息** | 鼠标悬停/长按地块 | 显示土壤湿度、作物进度 | 弹出信息面板 |

### 3.3 多人协作流程

```
玩家A在地块(3,5)种下了小麦
    ↓
玩家B看到后可以：
    ├─ 帮A浇水 → A的作物加速生长 → B获得友情点奖励
    ├─ 在相邻地块种植 → 形成连作/轮作效果
    └─ 等待A收获后交易
    ↓
多人同时在场 → 触发"协作加成"
```

## 5. 作物生长可视化

### 5.1 生长阶段表现

| 阶段 | 图标 | 视觉表现 | 可交互 |
|------|------|---------|--------|
| 种子 | 🌱 | 小绿芽，轻微摇摆动画 | 可浇水 |
| 幼苗 | 🌿 | 两片叶子，绿色较浅 | 可浇水 |
| 生长中 | 🌾 | 植株变大，颜色加深 | 可浇水 |
| 成熟 | 🌾✨ | 金黄色，发光特效 | 可收获 |

### 5.2 土壤状态可视化

| 湿度 | 颜色 | 表现 |
|------|------|------|
| 0-30% | 浅褐色 | 干燥裂纹 |
| 30-60% | 中褐色 | 正常状态 |
| 60-80% | 深褐色 | 湿润光泽 |
| 80-100% | 黑褐色 | 过湿，可能有水洼 |

## 6. 示例游戏流程（5分钟体验）

### 第0分钟：进入游戏
- 玩家输入名字"小明"
- 加入"公共农场"
- 初始金币：50

### 第1分钟：第一次种植
- 移动到空地(5,5)
- 种植小麦（花费2金币，剩余48）
- 地块显示 🌱

### 第2分钟：维护与等待
- 给小麦浇水（湿度+30%）
- 小麦进入生长加速状态
- 等待期间观察其他玩家操作

### 第3分钟：收获
- 小麦成熟（🌾✨发光）
- 点击收获
- 获得10金币（余额58）

### 第4分钟：升级种植
- 购买番茄种子（花费5金币，剩余53）
- 种植番茄
- 帮旁边玩家浇水（获得友情点）

### 第5分钟：社交互动
- 查看排行榜
- 与玩家B交易（用3个小麦换1个番茄种子）
- 计划下一轮种植布局

## 7. 进阶玩法（V2.0规划）

### 7.1 生态系统深化
- 连作惩罚：同一块地连续种同种作物 → 产量下降
- 轮作奖励：不同作物轮作 → 产量提升
- 益虫系统：种植花卉吸引蜜蜂 → 授粉加成

### 7.2 天气系统
- 晴天：正常生长
- 雨天：自动浇水，无需人工
- 干旱：生长减慢，需多浇水
- 虫害：随机事件，需手动除虫

### 7.3 经济系统
- 市场波动：作物价格随供需变化
- 期货交易：提前卖出未来收获
- 投资升级：购买自动灌溉系统

### 7.4 社交系统
- 农场访问：参观好友农场
- 互助任务：组队完成大订单
- 农场竞赛：限时活动，排名奖励

## 8. 技术实现要点（供DEV参考）

### 8.1 实时同步
- WebSocket 广播：玩家移动、作物状态变化
- 增量更新：只传输变化的数据
- 乐观更新：客户端先响应，服务端确认

### 8.2 状态管理
- 服务端权威：所有计算在服务端完成
- 客户端预测：移动等操作本地先执行
- 冲突处理：乐观锁 + 版本号

### 8.3 性能优化
- 地块只渲染可见区域
- 生长计算批量处理（每秒一次）
- 玩家离开视野停止同步
