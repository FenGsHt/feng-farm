## ⚖️ 经济平衡策略

### 新手期（0-7天）

**目标：** 快速积累启动资金

| 策略 | 说明 |
|------|------|
| 初始资金 | 200金币足够完成引导 |
| 引导奖励 | 前6步引导共获得约130金币 |
| 推荐循环 | 种小麦 → 出售 → 买更多小麦 → 扩展农田 |
| 预期收益 | 第1天可积累1000+金币 |

**经济公式：**
```
初始200 + 引导130 + 第1天种植收益(约700) = 1030金币
```

### 成长期（8-30天）

**目标：** 多元化收入，动物养殖

| 策略 | 说明 |
|------|------|
| 作物组合 | 小麦(快速) + 胡萝卜(中等) + 玉米(高收益) |
| 养殖起步 | 购买3-5只鸡，建立稳定产出 |
| 农田扩展 | 目标是解锁4块农田(1500金币) |
| 预期收入 | 每天500-2000金币 |

**经济公式：**
```
作物收入(1000) + 鸡蛋收入(500) + 任务(300) = 1800金币/天
```

### 成熟期（30天+）

**目标：** 最大化收益率

| 策略 | 说明 |
|------|------|
| 高级作物 | 以草莓、番茄为主 |
| 养殖规模 | 鸡20只 + 羊5只 + 牛2只 |
| 装饰加成 | 风车 + 温室提升效率 |
| 预期收入 | 每天5000+金币 |

## 📈 数值平衡公式

### 基础产出公式

```
单作物收益 = 售价 - 成本
收益率 = (售价 - 成本) / 成本 × 100%
```

## ⚠️ 风险控制

### 通货膨胀预防

1. **金币 sinks（消耗点）**
   - 定期推出新装饰、新作物种子
   - 动物有生命周期（可选）
   - 维护费用（可选，高级功能）

2. **产出上限**
   - 仓库容量上限：1000格
   - 农田/畜棚扩展需要大量金币

### 玩家流失预防

1. **保底机制**
   - 种子免费送（每天1次）
   - 任务失败仍获参与奖

2. **新手保护**
   - 前7天收益+20%
   - 引导期间免费种子无限量

| 情况 | 调整方案 |
