# 疯狂农场大亨 - 游戏策划概要

## 📱 游戏概述

| 项目 | 内容 |
|------|------|
| 游戏名称 | 疯狂农场大亨 (Crazy Farm Tycoon) |
| 游戏类型 | 模拟经营 / 放置挂机 |
| 目标平台 | iOS / Android |
| 目标用户 | 休闲玩家、模拟经营爱好者 |

---

## 🎮 核心玩法

### 1. 农田种植

- **作物种类**：小麦、胡萝卜、玉米、番茄、草莓
- **种植流程**：购买种子 → 种植到农田 → 等待生长 → 收获 → 出售
- **生长时间**：60秒 ~ 600秒（不同作物不同）

### 2. 动物养殖

| 动物 | 解锁条件 | 产出 |
|------|----------|------|
| 鸡 | 初始解锁 | 鸡蛋（60秒） |
| 羊 | 500金币 | 羊毛（300秒） |
| 牛 | 2000金币 | 牛奶（600秒） |

### 3. 商店系统

- 购买种子、动物、装饰物
- 扩建农田、畜棚

### 4. 任务系统

- **日常任务**：每日刷新，产出金币
- **成就任务**：一次性，完成获大奖
- **主线任务**：引导游戏进程

### 5. 社交系统

- 好友互访
- 排行榜竞争

---

## 💡 设计理念

1. **简单上手**：3分钟理解核心循环
2. **放置收益**：离线也能获得收益
3. **成长反馈**：每次升级有明确成就感
4. **社交互动**：好友系统增加粘性
