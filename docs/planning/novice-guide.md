# 新手引导流程设计

## 📌 设计目标

1. **降低上手门槛**: 让首次接触游戏的玩家也能快速理解核心玩法
2. **传递核心价值**: 3分钟内让玩家体验"种植→收获→赚钱→扩展"的正反馈循环
3. **提升早期留存**: 通过清晰的成长路径引导玩家度过迷茫期

---

## 🎮 引导流程总览

| 阶段 | 目标 | 耗时 | 解锁内容 |
|------|------|------|----------|
| 欢迎阶段 | 创建身份 | 30秒 | 角色命名 |
| 教程阶段 | 学会种植 | 60秒 | 农田、解锁种子 |
| 收获阶段 | 体验收益 | 60秒 | 出售作物、获得金币 |
| 扩展阶段 | 了解升级 | 60秒 | 商店、仓库、动物 |

**总时长**: 约 3-5 分钟

---

## 📝 详细流程

### 阶段一: 欢迎阶段 (30秒)

```
[开场动画]
    ↓
[欢迎界面] "欢迎来到疯狂农场大亨！"
    ↓
[输入名字] "请给你的农场起个名字吧"
    ↓
[获得新手礼包]
  - 500金币初始资金
  - 5袋小麦种子
  - 1块农田解琐
```

**UI要素**:
- 简洁的农场背景图
- 大字体的欢迎文案
- 名字输入框（带默认建议名）

---

### 阶段二: 教程阶段 (60秒)

```
[指引弹窗] "让我们来种植第一块农田吧！"
    ↓
[点击农田] → 弹出种植菜单
    ↓
[选择小麦] → 显示小麦信息（成本:10金币, 生长时间:30秒, 售价:25金币）
    ↓
[确认种植] → 种子减少1袋，农田显示生长状态
    ↓
[等待成熟] → 显示倒计时 "预计30秒后收获"
```

**引导策略**:
- 使用高亮边框突出当前操作对象
- 每步完成后自动进入下一步
- 跳过按钮始终可见（但默认不推荐）

---

### 阶段三: 收获阶段 (60秒)

```
[弹出提示] "小麦成熟啦！点击收获"
    ↓
[点击收获] → 获得5个小麦，显示 "+5 小麦"
    ↓
[指引出售] "去商店卖掉它们吧"
    ↓
[点击商店] → 显示收购商界面
    ↓
[出售小麦] → 获得25金币（5×5）
    ↓
[庆祝动画] ✨ "恭喜！你在农场赚到了第一桶金！"
```

**正反馈设计**:
- 收获时播放愉悦的音效
- 金币入账时显示 "+25" 飘字动画
- 首次出售解锁"出售"快捷按钮

---

### 阶段四: 扩展阶段 (60秒)

```
[解锁新功能] "现在你可以购买更多种子了"
    ↓
[商店界面] → 高亮显示"种子"标签
    ↓
[购买玉米] → 消耗30金币，获得玉米种子
    ↓
[指引仓库] "收获的作物会存入仓库"
    ↓
[仓库提示] 显示当前存储空间与容量
    ↓
[引导完成] "去建造你的梦想农场吧！🎉"
```

## 🔧 引导机制设计

### 1. 强制 vs 自由引导

| 阶段 | 模式 | 说明 |
|------|------|------|
| 欢迎+教程 | 强制引导 | 锁定其他UI元素，仅显示当前步骤 |
| 收获+扩展 | 半强制 | 可点击其他区域但会弹出指引提示 |
| 完成后 | 自由模式 | 玩家可自由探索 |

### 2. 引导状态存储

```javascript
// 引导进度存储
{
  "guide_completed": true,
  "guide_step": 4,
  "unlocked_features": [
    "planting",    // 种植
    "harvesting",  // 收获
    "selling",     // 出售
    "store",       // 商店
    "warehouse"    // 仓库
  ]
}
```

### 3. 跳过机制

- 首次进入：强制完成前3个阶段
- 第2-5次进入：可跳过教程直接开始
- 5次后：完全跳过，新玩家直接进入游戏

---

## 📊 引导数据埋点

| 事件 | 参数 | 用途 |
|------|------|------|
| guide_start | source | 引导开始来源 |
| guide_step_complete | step, duration | 各步骤完成率 |
| guide_skip | step | 引导跳过分析 |
| guide_complete | total_time | 引导完成总时长 |
| first_plant | crop_type | 首次种植分析 |
| first_sell | crop_type, amount | 首次出售分析 |

## 🎨 UI设计规范

### 引导高亮样式

```
边框: 3px solid #FFD700 (金色)
阴影: 0 0 15px rgba(255, 215, 0, 0.5)
动画: pulse (呼吸闪烁) 1s infinite
```

### 指引箭头

- 位置: 指向当前操作对象
- 颜色: 白色带描边
- 动画: 左右摆动 0.5s

### 弹窗样式

```
背景: rgba(0, 0, 0, 0.7)
圆角: 12px
内边距: 20px
按钮: 主色填充，次要操作居中显示
```

---

## ✅ 验收标准

1. **完成率**: 引导完成率 ≥ 90%
2. **流失点**: 引导中途流失 < 5%
3. **体验时间**: 3-5分钟完成全部引导
4. **功能测试**: 每个步骤可正常跳转，无卡死
5. **兼容性**: 支持手机/平板横竖屏
