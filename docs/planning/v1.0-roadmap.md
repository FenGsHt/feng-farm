## 🎯 新手引导流程 (v1.0)

### 引导目标
- 降低新玩家上手门槛
- 快速传递核心玩法
- 提升早期留存率

### 引导流程设计

```
进入游戏 → 命名角色 → 领取新手礼包 → 
完成教程任务 → 首次收获 → 首次出售 → 解锁更多功能
```

详细设计见 [novice-guide.md](./novice-guide.md)

## 💰 经济系统平衡 (v1.0)

### 金币产出

| 来源 | 产出方式 | 初期数量 |
|------|----------|----------|
| 初始资金 | 玩家起始金币 | 500 |
| 作物出售 | 出售收获的作物 | 视作物而定 |
| 任务奖励 | 完成每日/成就任务 | 50-500 |
| 出售动物 | 出售养殖的动物 | 视动物而定 |

### 金币消耗

| 用途 | 消耗方式 | 初期价格 |
|------|----------|----------|
| 购买种子 | 播种必需 | 10-50 |
| 购买动物 | 养殖必需 | 100-500 |
| 仓库扩建 | 扩大存储空间 | 200-1000 |
| 购买道具 | 加速/保护道具 | 20-200 |

### 平衡策略

1. **初期宽松**: 新手期金币产出略高于消耗，保证体验流畅
2. **中期调整**: 引入更多消耗点（金库扩建、高级种子）
3. **长期回收**: 通过商店升级、装饰物等回收多余金币

详细设计见 [economic-balance.md](./economic-balance.md)
