## 需求描述

### 一句话说明
玩家种植作物后，作物会随时间自动生长，经历多个阶段直至成熟可收获。

### 详细说明
1. **背景**: 种田游戏的核心体验是看着作物从种子成长为成熟作物
2. **目标**: 提供清晰的生长反馈，让玩家有成就感和期待感
3. **用户场景**: 玩家种下一颗种子，定期回来查看生长进度，最终收获

## 功能设计

### 作物类型

| 作物ID | 名称 | 图标 | 生长时间(秒) | 种子价格(金币) | 收获价格(金币) | 产量 |
|--------|------|------|-------------|---------------|---------------|------|
| wheat | 小麦 | 🌾 | 30 | 2 | 10 | 1 |
| tomato | 番茄 | 🍅 | 60 | 5 | 25 | 1 |
| corn | 玉米 | 🌽 | 120 | 12 | 60 | 1 |
| carrot | 胡萝卜 | 🥕 | 45 | 4 | 18 | 1 |
| pumpkin | 南瓜 | 🎃 | 180 | 20 | 100 | 1 |

### 生长阶段

每个作物经历 **4个生长阶段**:

| 阶段 | 名称 | 进度范围 | 视觉表现 | 可交互 |
|------|------|---------|---------|--------|
| 0 | 种子 | 0% - 19% | 🌱 小绿芽 | 可浇水 |
| 1 | 幼苗 | 20% - 59% | 🌿 两片叶子 | 可浇水 |
| 2 | 生长期 | 60% - 99% | 🌾/🍅/🌽 作物专属图标(小) | 可浇水 |
| 3 | 成熟 | 100% | ✨🌾/🍅/🌽 发光特效 | 可收获 |

### 生长计算

**基础公式:**
```
当前进度 = (当前时间 - 种植时间) / 总生长时间
生长阶段 = floor(当前进度 * 4)  [取值 0,1,2,3]
```

**浇水加速:**
- 地块浇水后，获得 **"湿润"** 状态（持续30秒）
- 湿润状态下，生长速度 **+50%**
- 多次浇水不叠加，只刷新持续时间

**生长速度倍率表:**

| 状态 | 倍率 | 说明 |
|------|------|------|
| 正常 | 1.0x | 基础速度 |
| 湿润 | 1.5x | 浇水后加速 |
| 干旱 | 0.5x | 土壤湿度<30%时 |
| 肥沃 | 1.2x | 土壤肥力>80%时 |

**实际生长时间计算:**
```
实际时间 = 基础时间 / (速度倍率1 * 速度倍率2 * ...)
```

示例:
- 小麦基础30秒，湿润状态: 30/1.5 = **20秒**
- 玉米基础120秒，湿润+肥沃: 120/(1.5*1.2) = **67秒**

### 作物状态数据

```javascript
{
  cropType: "wheat",      // 作物类型
  plantedAt: 1711045123,  // 种植时间戳(毫秒)
  stage: 2,               // 当前阶段 0-3
  growth: 75.5,           // 精确进度 0-100
  isWatered: true,        // 是否已浇水
  isReady: false,         // 是否可收获
  quality: 1.0            // 品质系数(影响售价)
}
```

### 品质系统(V1.1)

收获时计算品质，影响最终售价:

```
品质 = 基础品质 
     + 肥力加成(肥力/100*0.2) 
     + 浇水加成(浇水次数*0.05)
     - 干旱惩罚(曾经干旱次数*0.1)
```

品质等级:
- 0.8-1.0: 普通 (售价*1.0)
- 1.0-1.3: 优质 (售价*1.2)
- 1.3-1.6: 精品 (售价*1.5)
- 1.6+: 极品 (售价*2.0)

## 边界情况

1. **重复种植**: 已有作物的地块不能再次种植，提示"请先收获"
2. **离线生长**: 作物在玩家离线时继续生长（服务端计算）
3. **精确度**: 生长进度保留1位小数，阶段切换floor计算
4. **加速上限**: 最大加速不超过2.0x（防止过快）

## 接口设计

### 客户端 → 服务端

```javascript
// 种植
socket.emit('plant', { cropType: 'wheat' })

// 浇水
socket.emit('water')

// 收获
socket.emit('harvest')
```

### 服务端 → 客户端

```javascript
// 游戏状态更新（包含所有作物状态）
socket.emit('game-state', {
  plots: [
    { x: 0, y: 0, crop: { cropType: 'wheat', stage: 2, growth: 75.5, isWatered: true }}
  ]
})

// 操作结果
socket.emit('action-result', {
  action: 'harvest',
  success: true,
  reward: 12,  // 实际获得金币（已计算品质）
  quality: 1.2
})
```
