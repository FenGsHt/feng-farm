## 需求描述

### 一句话说明
农场会随时间变化出现不同天气，影响作物生长和玩家操作。

### 详细说明
1. **背景**: 真实种田受天气影响，增加策略性和不可预测性
2. **目标**: 让玩家需要关注天气变化，调整种植策略
3. **用户场景**: 雨天自动浇水，玩家可以偷懒；干旱需要紧急浇水

## 功能设计

### 天气类型

| 天气 | 图标 | 持续时间 | 出现概率 | 对作物影响 |
|------|------|---------|---------|-----------|
| ☀️ 晴天 | ☀️ | 60秒 | 40% | 正常生长 |
| 🌧️ 雨天 | 🌧️ | 40秒 | 25% | 自动浇水，生长+10% |
| ☁️ 多云 | ☁️ | 50秒 | 20% | 蒸发-50%，土壤保湿 |
| 🌙 夜晚 | 🌙 | 30秒 | 10% | 生长暂停（无光照） |
| 🔥 干旱 | 🔥 | 30秒 | 5% | 土壤快速干燥，生长-50% |

### 天气循环

天气切换间隔: 30-60秒随机，根据概率随机选择

**天气状态数据:**
```javascript
{
  current: "sunny",      // 当前天气
  startedAt: 1711045123, // 开始时间
  duration: 60,          // 持续秒数
  nextChange: 1711045183 // 下次切换时间
}
```

### 天气效果详解

#### ☀️ 晴天 (Sunny)
- 生长速度 1.0x
- 土壤正常蒸发 -5%/分钟
- 策略: 正常种植，记得浇水

#### 🌧️ 雨天 (Rainy)
- 生长速度 1.1x (+10%)
- 土壤自动浇水，湿度+30%/分钟
- 策略: 无需手动浇水，趁机种植

#### ☁️ 多云 (Cloudy)
- 生长速度 0.9x (-10%)
- 蒸发减慢 -2%/分钟
- 策略: 土壤保湿时间长

#### 🌙 夜晚 (Night)
- 生长速度 0x (暂停)
- 策略: 休息时段

#### 🔥 干旱 (Drought)
- 生长速度 0.5x (-50%)
- 土壤快速干燥 -10%/分钟
- 策略: 紧急浇水
