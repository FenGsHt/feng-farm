## 需求描述

### 一句话说明
农场会出现各类昆虫，包括授粉益虫和破坏害虫，为玩家提供额外策略维度。

### 详细说明
1. **背景**: 真实农场生态中存在昆虫互动，增加生物多样性和策略选择
2. **目标**: 让玩家需要关注昆虫状态，利用益虫加成，防治害虫损失
3. **用户场景**: 种植花卉吸引蜜蜂获得授粉加成；发现害虫需及时手动清除

## 功能设计

### 昆虫类型

#### 益虫

| 昆虫 | 图标 | 出现条件 | 效果 | 持续时长 |
|------|------|---------|------|---------|
| 🐝 蜜蜂 | 🐝 | 种植花卉类作物 | 周围3x3范围作物生长+20% | 60秒 |
| 🦋 蝴蝶 | 🦋 | 种植3种以上不同作物 | 作物品质+0.1 | 90秒 |
| 🐞 瓢虫 | 🐞 | 有害虫存在时概率出现 | 自动消灭1只害虫 | 单次 |

#### 害虫

| 昆虫 | 图标 | 出现条件 | 负面效果 | 传播方式 |
|------|------|---------|---------|---------|
| 🐛 蚜虫 | 🐛 | 雨天概率出现 | 作物生长-30% | 向相邻作物传播 |
| 🐌 蜗牛 | 🐌 | 湿度>80%时出现 | 成熟作物被啃食，产量-50% | 缓慢移动 |
| 🦗 蝗虫 | 🦗 | 干旱天气概率爆发 | 整片区域作物受损 | 快速扩散 |

### 昆虫行为机制

#### 益虫生成

```
蜜蜂出现率 = 花卉数量 × 10% （最高50%）
蝴蝶出现率 = 作物种类数 × 15% （最高45%）
瓢虫出现率 = 害虫数量 × 25% （最高75%）
```

**触发检查**: 每30秒检查一次，满足条件则在随机地块生成

#### 害虫生成

```
蚜虫出现率(雨天) = 20%
蜗牛出现率(高湿) = 湿度% × 0.3
蝗虫出现率(干旱) = 15%
```

**传播机制**:
- 蚜虫: 每20秒向相邻作物传播，传播概率50%
- 蜗牛: 每15秒移动1格，优先前往成熟作物
- 蝗虫: 每10秒扩散1格，影响区域内所有作物

### 玩家交互

| 操作 | 触发方式 | 效果 | 消耗 |
|------|---------|------|------|
| **除虫** | 点击有害虫的地块 | 消灭该害虫 | 无 |
| **吸引蜜蜂** | 种植花卉（🌻向日葵、🌷郁金香） | 蜜蜂出现率+30% | 种子金币 |
| **放置诱虫灯** | 购买并放置道具 | 害虫吸引至灯下，便于集中处理 | 50金币 |

## 验收标准

| 验收项 | 验收标准 | 优先级 |
|--------|---------|--------|
| 蜜蜂生成 | 种植花卉后30秒内蜜蜂出现概率>30% | P0 |
| 蜜蜂效果 | 蜜蜂周围3x3范围作物生长速度提升20% | P0 |
| 蚜虫传播 | 蚜虫每20秒向相邻作物传播，概率50% | P0 |
| 除虫操作 | 点击有害虫地块可清除害虫，有视觉反馈 | P0 |
| 害虫影响 | 受蚜虫影响作物生长速度减少30% | P0 |
| 昆虫图标 | 益虫和害虫显示不同图标动画 | P1 |
| 蝗虫爆发 | 干旱天气有15%概率触发蝗虫事件 | P1 |
| 昆虫统计 | 界面显示当前益虫/害虫数量 | P2 |

## 边界情况

1. **益虫上限**: 同类型益虫最多3只，超出不生成
2. **害虫上限**: 单地块最多1只害虫，满则向相邻传播
3. **作物收割**: 收割后该地块昆虫自动消失
4. **离线处理**: 玩家离线期间昆虫正常计时，过期消失

- **受保护**: 作物边缘金色光圈（有益虫效果）

## 接口设计

### 客户端 → 服务端

```javascript
// 除虫
socket.emit('remove-pest', { x: 5, y: 5 })

// 放置诱虫灯
socket.emit('place-trap', { x: 3, y: 3 })
```

### 服务端 → 客户端

```javascript
// 昆虫生成通知
socket.emit('insect-spawn', {
  insectId: "bee_001",
  type: "bee",
  position: { x: 5, y: 5 }
})

// 昆虫消失通知
socket.emit('insect-despawn', { insectId: "bee_001" })

// 昆虫效果更新
socket.emit('insect-effect', {
  insectId: "bee_001",
  affectedPlots: [{ x: 4, y: 5 }, { x: 5, y: 4 }, { x: 6, y: 5 }],
  effect: "growth_boost_20"
})
```
