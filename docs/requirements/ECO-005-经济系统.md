- **需求ID**: ECO-005

- **提出时间**: 2026-03-22
- **提出人**: PD
- **状态**: 进行中

## 需求描述

### 一句话说明
玩家通过种植、交易、投资等经济活动积累金币，实现农场扩张和效率提升。

### 详细说明
1. **背景**: 经济循环是游戏的核心驱动力，让玩家有明确的成长目标
2. **目标**: 提供多样化的收入来源和消费途径，保持经济平衡
3. **用户场景**: 玩家收获作物卖钱，购买更好种子，投资自动化设施，与其他玩家交易

## 功能设计

### 货币系统

**主货币: 金币 (💰)**
- 初始值: 50金币
- 上限: 无上限
- 用途: 购买种子、道具、设施

**副货币: 友情点 (❤️)**
- 获取: 帮助其他玩家（浇水、除虫）
- 用途: 兑换特殊种子、装饰道具
- 上限: 9999

### 收入来源

| 来源 | 计算方式 | 说明 |
|------|---------|------|
| **作物销售** | 基础价格 × 品质倍率 | 主要收入来源 |
| **帮助他人** | 5-15友情点 | 浇水、除虫 |
| **每日登录** | 20金币 | 每日首次登录奖励 |
| **成就奖励** | 50-500金币 | 完成成就任务 |
| **交易收入** | 协商价格 | 与其他玩家交易 |

### 支出项目

| 项目 | 价格 | 说明 |
|------|------|------|
| 🌾 小麦种子 | 2金币 | 基础作物 |
| 🍅 番茄种子 | 5金币 | 中级作物 |
| 🌽 玉米种子 | 12金币 | 高级作物 |
| 💧 超级肥料 | 15金币 | 肥力+50% |
| 🛡️ 驱虫剂 | 10金币 | 清除3x3范围害虫 |
| 💡 诱虫灯 | 50金币 | 吸引害虫便于清除 |
| 🚿 自动灌溉 | 500金币 | 单地块自动浇水（一次性） |
| 🏗️ 扩建地块 | 100×等级金币 | 增加可种植地块 |

### 市场波动系统

**价格波动机制:**

```
当前价格 = 基础价格 × 市场倍率

市场倍率 = 1.0 + sin(时间周期) × 0.3 + 供需影响

供需影响 = -0.2 × (该作物销售量 / 平均销量 - 1)
```

**价格区间:**
| 市场状态 | 倍率范围 | 图标 | 说明 |
|---------|---------|------|------|
| 暴跌 | 0.5 - 0.7 | 📉 | 供过于求 |
| 低迷 | 0.7 - 0.9 | ⬇️ | 销量一般 |
| 正常 | 0.9 - 1.1 | ➡️ | 标准价格 |
| 上涨 | 1.1 - 1.3 | ⬆️ | 需求增加 |
| 暴涨 | 1.3 - 1.5 | 📈 | 供不应求 |

**更新频率**: 每5分钟更新一次市场倍率

### 交易行系统

**挂单交易:**

```javascript
// 挂单结构
{
  orderId: "order_001",
  seller: "playerA",
  item: "wheat",
  quantity: 10,
  unitPrice: 12,      // 单价
  totalPrice: 120,    // 总价
  postedAt: 1711045123,
  expiresIn: 3600     // 1小时后过期
}
```

**交易类型:**
| 类型 | 说明 | 手续费 |
|------|------|--------|
| 现货交易 | 即时买卖当前作物 | 无 |
| 挂单出售 | 设定价格等待买家 | 5% |
| 求购订单 | 设定价格等待卖家 | 5% |

**交易限制:**
- 每人最多5个活跃订单
- 最低价格: 基础价格的50%
- 最高价格: 基础价格的200%

### 投资升级系统

**农场设施:**

| 设施 | 价格 | 效果 | 解锁条件 |
|------|------|------|---------|
| 🚿 自动灌溉 | 500金币 | 单地块自动保持湿度>60% | 拥有10块地 |
| 🏚️ 仓库 | 1000金币 | 收获作物自动存储，可批量出售 | 等级3 |
| 🌡️ 温室 | 2000金币 | 3x3区域无视天气影响 | 等级5 |
| 🐝 蜂箱 | 800金币 | 永久吸引蜜蜂，无需花卉 | 等级4 |
| 🔬 实验室 | 3000金币 | 解锁高级种子（金色品质） | 等级7 |

**升级路径:**
```
小型农场(12格) → 中型农场(24格) → 大型农场(48格) → 超级农场(96格)
     ↓                ↓                ↓                 ↓
  100金币         500金币          2000金币          10000金币
```

### 玩家等级系统

**经验值获取:**
| 行为 | 经验值 |
|------|--------|
| 种植作物 | 1 XP |
| 收获作物 | 3 XP |
| 帮助好友 | 5 XP |
| 完成交易 | 10 XP |
| 解锁设施 | 50 XP |

**等级奖励:**
| 等级 | 称号 | 解锁 |
|------|------|------|
| 1 | 新手农夫 | 基础作物 |
| 3 | 熟练农夫 | 仓库设施 |
| 5 | 专业农夫 | 温室设施 |
| 7 | 农场主 | 实验室 |
| 10 | 农业大亨 | 超级种子 |

## 验收标准

| 验收项 | 验收标准 | 优先级 |
|--------|---------|--------|
| 初始金币 | 新玩家进入获得50金币 | P0 |
| 作物售价 | 收获小麦获得10金币（基础价格） | P0 |
| 市场波动 | 价格每5分钟变化，波动范围±50% | P0 |
| 价格波动UI | 界面显示当前市场状态图标 | P0 |
| 交易行 | 玩家可挂单出售作物，其他玩家可购买 | P1 |
| 自动灌溉 | 购买后单地块自动保持湿度 | P1 |
| 扩建农场 | 支付金币增加可种植地块数 | P1 |
| 等级系统 | 玩家获得经验升级，解锁新内容 | P1 |
| 友情点 | 帮助其他玩家获得友情点，可兑换道具 | P2 |

## 边界情况

1. **金币不足**: 购买时金币不足，提示余额不足并显示差额
2. **价格下限**: 市场倍率最低0.5，防止价格崩盘
3. **交易过期**: 挂单1小时后自动下架，作物返回背包
4. **等级封顶**: 最高等级10级，经验继续累计但不升级
